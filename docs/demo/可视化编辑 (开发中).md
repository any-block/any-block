### tab

:::mditTabs-e

@tab linux

可以通过执行以下命令在终端中使用 apt 包安装程序：

```js
apt-get install python3.6 // 根据你具体的版本而定
```

安装后即可

@tab windows

- 转到官方 Python 站点，并导航到最新版本
  > 在撰写本文时，为 `3.10.6`
- 下载适用于您平台的二进制文件。执行二进制。
- 除了将 Python 添加到 `PATH` 之外，您不需要选择任何选项，因为默认安装程序具有您需要的一切。只需单击“安装”即可。

@tab macOS

- 转到官方 Python 站点，并导航到最新版本
  > 在撰写本文时，为 `3.10.6`
- 下载适用于您平台的二进制文件。执行二进制。
- 在 Mac 上，这将默认在 dmg 安装程序中完成。

:::
