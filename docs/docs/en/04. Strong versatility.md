# AnyBlock Powerful Versatility

AnyBlock is a lightweight, highly versatile TypeScript library, making it suitable for use in various environments.

Additionally, for different SSGs (Static Site Generators), it provides encapsulated plugins for these independent frameworks to facilitate user adoption, along with numerous demos for reference.

Current Adaptations of AnyBlock:

- [x] Obsidian
    - [x] Live Preview Mode (CodeMirror)
    - [x] Reading Mode (Html)
- [x] CodeMirror
- [x] Markdown-it System
    - [x] Vuepress    | [Tutorial/Demo](https://github.com/any-block/VuepressDemo)
    - [x] Vitepress   | [Tutorial/Demo](https://github.com/any-block/VitePressDemo)
- Remark/Unified System | (Some limitations exist, enhancements in progress)
    - [ ] Astro
    - [x] Quartz4     | [Tutorial/Demo](https://github.com/any-block/QuartzDemo)
    - [x] Docusaurus  | [Tutorial/Demo](https://github.com/any-block/DocusaurusDemo)
- [x] Others, Web Version | [Demo](https://any-block.github.io/any-block/)
