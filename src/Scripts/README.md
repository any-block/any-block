# AnyBlock Tools / Scripts (Developing)

一些 anyblock 的独立工具和脚本 (开发中)

包括:

- 检测: 检测单篇笔记/所有库中所有的 AnyBlock 区域
  这里也可以帮助用户检查一下有没有误识别
- 修改: 在检测的基础上，可以:
  - 去除所有 anyblock 的附加选择器标识，如 `:::`、如 `[...]`
  - 正则修改他们
  - 将普通选择器转换为代码块等更稳定的选择器
  - 将渲染结果转为图片
  此外，修改可以选用不同策略: 覆盖 / 替换并备份 / 生成修改后的副本
- 一些应用:
  - 如在 obsidian 导出 pdf 或其他格式时，用于网络博客时，其他等等其他环境中，可以转换为更稳定的代码块形式

在 obsidian 版本中会自带这些工具
